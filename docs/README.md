# ブログシステム（課題用）

このプロジェクトは、Laravel 12 を使用した認証機能付きブログシステムです。

## 1. プロジェクト概要

ユーザーが記事を投稿・管理できるブログプラットフォーム。

## 2. 主な機能

- **認証機能**: 会員登録、ログイン、ログアウト
- **記事管理**: 新規投稿、一覧表示、詳細閲覧
- **マイページ**: 自分の投稿履歴の確認
- **編集・削除**: 自分が書いた記事のみ編集・削除が可能

## 3. 要件定義（詳細）

### ユーザー機能

- ログインしていないユーザー：記事の一覧と詳細のみ閲覧可能
- ログイン済みのユーザー：記事の投稿、マイページの閲覧、自分の記事の編集・削除が可能

### データベース構成

- **users テーブル**: ユーザー名、メール、パスワード
- **posts テーブル**: タイトル、本文、ユーザー ID（外部キー）、作成日時

  データベース設計の詳細は、以下の設計書を確認してください。

* [データベース設計書](./docs/db_design.md)

## 4. 開発環境

- PHP 8.x (Laravel 12)
- MySQL 8.0
- Docker / Docker Compose
- CI: GitHub Actions

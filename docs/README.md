# 操作マニュアル

本システムは、記事の閲覧・投稿・管理を行うことができるブログプラットフォームです。

---

## 1. 記事を閲覧する（全ユーザー）

ログインしていない状態でも、以下の操作が可能です。

- **記事一覧の閲覧**  
  トップ画面で投稿された記事の最新一覧を確認できます。

- **記事詳細の閲覧**  
  記事のタイトルをクリックすると、本文全文を閲覧できます。

---

## 2. 会員登録・ログイン

投稿やマイページ機能を利用するには、アカウントが必要です。

- **新規登録**  
  ヘッダーの「Register」から、名前・メールアドレス・パスワードを入力して登録します。

- **ログイン**  
  登録済みのメールアドレスとパスワードでログインします。

---

## 3. 記事を投稿する（ログインユーザーのみ）

1. ログイン後、ダッシュボードまたはヘッダーの「新規投稿」ボタンをクリックします。
2. **タイトル** と **本文** を入力します。
3. **保存ボタン** を押すと記事が公開されます。

---

## 4. 自分の記事を管理する（マイページ）

ログイン後、マイページ（ダッシュボード）から自分の投稿を一覧で確認できます。

- **編集**  
  自分の記事一覧にある「編集」リンクから内容を修正できます。

- **削除**  
  「削除」ボタンを押すと確認ダイアログが表示され、承認すると記事が削除されます。

⚠ **注意**  
他のユーザーが投稿した記事を編集・削除することはできません。

# 🛠 開発者向け情報

## 主なルート一覧

| 機能               | URL                | ルート名        | 権限             |
| ------------------ | ------------------ | --------------- | ---------------- |
| トップ（記事一覧） | `/`                | `posts.index`   | 全員             |
| 記事詳細           | `/posts/{id}`      | `posts.show`    | 全員             |
| 会員登録           | `/register`        | `register`      | ゲストのみ       |
| ログイン           | `/login`           | `login`         | ゲストのみ       |
| ダッシュボード     | `/dashboard`       | `dashboard`     | ログインユーザー |
| 記事作成           | `/posts/create`    | `posts.create`  | ログインユーザー |
| 記事編集           | `/posts/{id}/edit` | `posts.edit`    | 投稿者本人       |
| 記事削除           | `/posts/{id}`      | `posts.destroy` | 投稿者本人       |
